## Calculator Project
with this code you can operate simple calculations
## How to use it
1.Input numbers using the buttons provided.
2.Use the arithmetic operation buttons to perform calculations.
3.Click "=" to see the result.
## Features
Addition(+)
Substraction(-)
multiplication(*)
division (/)
